# Practicum_projects

Сборник проектов по анализу данных, включающий: исследование потребительских трендов, сегментацию аудитории и прогнозирование временных рядов. Цель — отработка методов извлечения инсайтов из данных для принятия бизнес-решений.

## Структура репозитория

| Название проекта / Папка | Технологии | Краткое описание |
| --- | --- | --- |
| **Дашборды** | Визуализация данных | Папка с дашбордами для визуализации аналитических данных. |
| **Кто начитаннее (часть 1)** | SQL | Анализ данных с помощью SQL-запросов для выявления читательских предпочтений (первая часть кейса). |
| **Кто начитаннее (часть 2) и битва интерфейсов** | Python | Продолжение кейса: обработка данных на Python и создание пользовательского интерфейса для сравнения результатов. |
| **Недвижимость — где спрос, когда продавать** | SQL | Исследование рынка недвижимости: анализ спроса по локациям и оптимальное время для продажи с помощью SQL-аналитики. |
| **Новая лента — проверка боем** | Python | Проект по проверке гипотез или A/B-тестированию функционала «новой ленты» с использованием инструментов Python. |
| **Осенний парадокс онлайн-продаж билетов** | Python | Анализ сезонных аномалий в продажах билетов онлайн: выявление причин «осеннего парадокса» с помощью обработки данных на Python. |

> **Примечание:** В корне также находятся служебные файлы `.gitignore` и `README.md`.

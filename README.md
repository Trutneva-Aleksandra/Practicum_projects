# Practicum_projects

Сборник проектов по анализу данных, включающий: исследование потребительских трендов, сегментацию аудитории и прогнозирование временных рядов. Цель — отработка методов извлечения инсайтов из данных для принятия бизнес-решений.

## Структура репозитория

| Название проекта / Папка | Технологии | Краткое описание |
| --- | --- | --- |
| [Дашборды](Дашборды/) | Визуализация данных | Папка с ссылками на дашборды для визуализации аналитических данных. |
| [Кто начитаннее (часть 1)](Кто%20начитаннее%20%28часть%201%29%20%28SQL%29/) | SQL | Проект посвящён аналитике сервиса электронных и аудиокниг на примере пользователей Москвы и Санкт‑Петербурга. Основная задача — рассчитать ключевые метрики (MAU, Retention, LTV и др.) с помощью SQL и интерпретировать результаты для поиска точек роста продукта (первая часть кейса). |
| [Кто начитаннее (часть 2) и битва интерфейсов](Кто%20начитаннее%20%28часть%202%29%20и%20битва%20интерфейсов%20%28Python%29/) | Python | 2 проекта, показывающих навыки проверки гипотез в Python. Предстояло проверить гипотезу, что пользователи из Санкт Петербурга проводят в среднем больше времени за чтением и прослушиванием книг в приложении, чем пользователи из Москвы (вторая часть кейса) и оценить влияние упрощённого интерфейса интернет-магазина на конверсию в покупку в течение 7 дней после регистрации. |
| [Недвижимость — где спрос, когда продавать](Недвижимость%20-%20где%20спрос,%20когда%20продавать%20%28SQL%29/) | SQL | Исследование рынка недвижимости: анализ спроса по локациям и оптимальное время для продажи с помощью SQL-аналитики. |
| [Новая лента — проверка боем](Новая%20лента%20-%20проверка%20боем%20%28Python%29/) | Python | Проект по проверке гипотез или A/B-тестированию функционала «новой ленты» с использованием инструментов Python. |
| [Осенний парадокс онлайн-продаж билетов](Осенний%20парадокс%20онлайн-продаж%20билетов%20%28Python%29/) | Python | Анализ сезонных аномалий в продажах билетов онлайн: выявление причин «осеннего парадокса» с помощью обработки данных на Python. |

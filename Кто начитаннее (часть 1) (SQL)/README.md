# Анализ поведения пользователей книжного сервиса: расчёт ключевых метрик

**Статус:** завершён
**Стек:** SQL (PostgreSQL), Jupyter Notebook, аналитика продуктовых метрик
**Данные:** обезличенные данные активности пользователей (часы, события, контент, гео) за 2024 год
**Цель:** глубже понять поведение пользователей и выявить закономерности взаимодействия с платформой

## О проекте

Проект посвящён продуктовой аналитике сервиса электронных книг и аудиокниг на примере пользователей Москвы и Санкт‑Петербурга. Основная задача — рассчитать ключевые метрики (MAU, Retention, LTV и др.) с помощью SQL и интерпретировать результаты для поиска точек роста продукта.

В рамках проекта смоделирована упрощённая монетизация (подписка 399 руб./мес.) и проведён когортный анализ удержания пользователей, привлечённых в ходе рекламной кампании.

## Задачи

- Рассчитать MAU авторов и произведений за ноябрь 2024, выявить лидеров по вовлечённости.
- Построить когортный Retention Rate для пользователей, активных 2 декабря 2024.
- Оценить средний LTV пользователей в Москве и Санкт‑Петербурге.
- Проанализировать динамику средней выручки на час активности (avg_hour_rev) за сентябрь–ноябрь 2024.
- Сформулировать выводы и практические гипотезы.

## Ключевые метрики и результаты

| Метрика | Что показывает | Ключевой вывод |
|---|---|---|
| **MAU авторов** | Охват аудитории по авторам | Лидер — Андрей Усачёв (7 107 MAU), явный драйвер вовлечённости; сильный сегмент детской литературы |
| **MAU произведений** | Популярность конкретных произведений | Серия «Собачка Соня» стабильно в топе; аудиоформат востребован в детском сегменте |
| **Retention Rate** | Удержание когорты по дням жизни | Первичное удержание 63% (день 1), затем плавное снижение; есть небольшой подъём в конце недели |
| **LTV** | Средний вклад пользователя в выручку | LTV в Москве (764.55 руб.) выше, чем в СПб (731.82 руб.) |
| **Avg_hour_rev** | Эффективность монетизации на единицу активности | Выручка на час снижается (61.7 → 51.04 руб.) при росте MAU — сигнал о «разбавлении» среднего новыми пользователями |

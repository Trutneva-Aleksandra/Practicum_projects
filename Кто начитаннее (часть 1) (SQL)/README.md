# Кто начитаннее (часть 1 - SQL).

Проект посвящён продуктовой аналитике сервиса электронных книг и аудиокниг на примере пользователей Москвы и Санкт‑Петербурга. Основная задача — рассчитать ключевые метрики (MAU, Retention, LTV и др.) с помощью SQL и интерпретировать результаты для поиска точек роста продукта.

**Дата:** 18.05.2026

**Статус:** завершён.

**Стек:** SQL (PostgreSQL).

**Данные:** обезличенные данные активности пользователей (часы, события, контент, гео) за 2024 год.

**Цель:** глубже понять поведение пользователей и выявить закономерности взаимодействия с платформой.

**Задачи:**
- Рассчитать MAU авторов и произведений за ноябрь 2024, выявить лидеров по вовлечённости.
- Построить когортный Retention Rate для пользователей, активных 2 декабря 2024.
- Оценить средний LTV пользователей в Москве и Санкт‑Петербурге.
- Проанализировать динамику средней выручки на час активности за сентябрь–ноябрь 2024.
- Сформулировать выводы и практические гипотезы.

**Ключевые метрики, результаты и рекомендации:**
| Метрика | Что показывает | Ключевой вывод | Рекомендации |
|---|---|---|---|
| **MAU авторов** | Охват аудитории по авторам | Лидер — Андрей Усачёв (7 107 MAU), явный драйвер вовлечённости; сильный сегмент детской литературы | (1) Создать персонализированные подборки по произведениям Андрея Усачёва для новых пользователей; (2) Запустить email-рассылки для фанатов детской литературы с новинками авторов схожего жанра; (3) Разработать программу лояльности для активных читателей детских книг |
| **MAU произведений** | Популярность конкретных произведений | Серия «Собачка Соня» стабильно в топе; аудиоформат востребован в детском сегменте | (1) Продвигать серию «Собачка Соня» через таргетированную рекламу для родителей дошкольников; (2) Создать тематические плейлисты аудиокниг по возрастным категориям; (3) Внедрить систему рекомендаций похожих произведений на основе популярности серии |
| **Retention Rate** | Удержание когорты по дням жизни | Первичное удержание 63% (день 1), затем плавное снижение; есть небольшой подъём в конце недели | (1) Внедрить push-уведомления на 2-3 день без активности с персональными рекомендациями; (2) Запустить weekend-челленджи для удержания пользователей в выходные; (3) Добавить систему достижений за регулярный возврат в приложение |
| **LTV** | Средний вклад пользователя в выручку | LTV в Москве (764.55 руб.) выше, чем в СПб (731.82 руб.) | (1) Провести анализ конверсии платных подписок в Москве и СПб для выравнивания показателей; (2) Разработать специальные предложения для Санкт-Петербурга (скидки, бонусы); (3) Внедрить программу семейного доступа для увеличения среднего чека |
| **Avg_hour_rev** | Эффективность монетизации на единицу активности | Выручка на час снижается (61.7 → 51.04 руб.) при росте MAU — сигнал о «разбавлении» среднего новыми пользователями | (1) Сегментировать пользователей по уровню активности и предложить разные тарифные планы; (2) Внедрить систему бонусов за длительное использование платформы; (3) Разработать премиум-контент для платных подписчиков с эксклюзивными материалами |

# Недвижимость: где спрос, когда продавать? (SQL)

Анализ рынка жилой недвижимости Санкт‑Петербурга и Ленинградской области. Анализ выполнен с помощью SQL (расчёт перцентилей, фильтрация выбросов, категоризация по длительности и месяцам, группировка и агрегация метрик).

**Дата:** 14.01.2026

**Статус:** завершён

**Стек:** SQL (PostgreSQL)

**Данные:** архив объявлений о продаже жилой недвижимости за 2014-2019 годы

**Цель:** выявить наиболее привлекательные сегменты недвижимости и оптимальные периоды для публикации объявлений, чтобы выстроить эффективную бизнес‑стратегию.

**Задачи:**
- Познакомиться с данными и проверить их качество (диапазон дат, основные статистики, выбросы).
- Проанализировать время активности объявлений: понять, какие сегменты «зависают» дольше, чем другие, и как это различается между Санкт‑Петербургом и Ленинградской областью.
- Выявить сезонные тенденции: определить месяцы с наибольшей активностью публикации и снятия объявлений.
- Сформулировать практические рекомендации для продавцов и покупателей.

**Что сделано:**
*   **Знакомство с данными.** Определён временной диапазон (2014–2019), оценены основные метрики (средняя и медианная стоимость за кв. м, площадь, количество комнат и др.), выявлен процент снятых с публикации объявлений (~86,55 %), доля объявлений по Санкт‑Петербургу (~66,47 %).
*   **Ad‑hoc задача 1: время активности объявлений.** С помощью CTE и оконных функций отсеяны аномальные значения (99‑й и 1‑й перцентили), объявления разбиты на категории по длительности экспозиции (до 1 месяца, до 3 месяцев и т. д.). Рассчитаны ключевые метрики в разрезе региона и времени активности (количество сделок, средняя цена за кв. м, медианы площади, комнат, этажа и др.).
*   **Ad‑hoc задача 2: сезонность.** Выделены месяцы публикации и снятия объявлений, построена сводная статистика по месяцам (число объявлений, средняя цена, средняя площадь). Проанализированы пики активности.
*   **Интерпретация и выводы.** На основе результатов сформулированы закономерности рынка и практические рекомендации.

**Выводы:**
- В Санкт‑Петербурге заметная доля объявлений активна более полугода (около 21 %); в Ленинградской области таких «долгоиграющих» объявлений меньше.
- В СПб дольше продаются более дорогие и крупные объекты; в Ленобласти быстрее уходят бюджетные варианты и компактные квартиры.
- Осень — удачное время для публикации объявлений; декабрь — пик продаж; летом рынок замедляется.
- Высокая стартовая цена не гарантирует быструю продажу.

**Рекомендации:**
- Продавцам в СПб: размещать объявления в октябре–ноябре, чтобы успеть к декабрьскому пику; для крупной недвижимости закладывать срок продажи более полугода.
- Продавцам в Ленобласти: не завышать цену — бюджетные объекты продаются быстрее; делать ставку на доступность.
- Покупателям: проявлять наибольшую активность в 4‑м квартале; летом можно снизить интенсивность поиска.
